      89             :  * Note that Figure 20 of \cite Buchman:2012dw illustrates additional

      91             :  * of Figure 20 indicates that there are additional spherical shells
      92             :  * outside the "CA" and "CB" blocks; `CylindricalBinaryCompactObject`
      93             :  * has these extra shells inside "EA" only if the option `IncludeInnerSphereA`
      94             :  * is true, it has the extra shells inside "EB" only if the option

      96             :  * "CA" and "CB" only if `IncludeOuterSphere` is true.
      97             :  * If the shells are absent, then the "EA" and "EB"
      98             :  * blocks extend to the excision boundaries and the "CA" and "CB" blocks
      99             :  * extend to the outer boundary.
     100             :  *
     101             :  * The Blocks are named as follows:
     102             :  * - Each of CAFilledCylinder, EAFilledCylinder, EBFilledCylinder,
     103             :  *   MAFilledCylinder, MBFilledCylinder, and CBFilledCylinder consists
     104             :  *   of 5 blocks, named 'Center', 'East', 'North', 'West', and
     105             :  *   'South', so an example of a valid block name is
     106             :  *   'CAFilledCylinderCenter'.
     107             :  * - Each of CACylinder, EACylinder, EBCylinder, and CBCylinder
     108             :  *   consists of 4 blocks, named 'East', 'North', 'West', and 'South',
     109             :  *   so an example of a valid block name is 'CACylinderEast'.
     110             :  * - The Block group called "Outer" consists of all the CA and CB blocks. They
     111             :  *   all border the outer boundary if `IncludeOuterSphere` is false.
     112             :  * - If `IncludeOuterSphere` is true, then there are more blocks named
     113             :  *   OuterSphereCAFilledCylinder, OuterSphereCBFilledCylinder,
     114             :  *   OuterSphereCACylinder, and OuterSphereCBCylinder.
     115             :  *   These are in a Block group called "OuterSphere",
     116             :  *   and all of these border the outer boundary.
     117             :  * - The Block group called "InnerA" consists of all the EA, and MA
     118             :  *   blocks. They all border the inner boundary "A" if
     119             :  *   `IncludeInnerSphereA` is false.
     120             :  * - If `IncludeInnerSphereA` is true, then there are new blocks
     121             :  *   InnerSphereEAFilledCylinder, InnerSphereMAFilledCylinder, and
     122             :  *   InnerSphereEACylinder. These are in a Block group called "InnerSphereA",
     123             :  *   and all of these border the inner excision boundary "A".
     124             :  * - The Block group called "InnerB" consists of all the EB, and MB
     125             :  *   blocks. They all border the inner boundary "B" if
     126             :  *   `IncludeInnerSphereB` is false.
     127             :  * - If `IncludeInnerSphereB` is true, then there are new blocks
     128             :  *   InnerSphereEBFilledCylinder, InnerSphereMBFilledCylinder, and
     129             :  *   InnerSphereEBCylinder. These are in a Block group called "InnerSphereB",
     130             :  *   and all of these border the inner excision boundary "B".
     131             :  *
     132             :  * If \f$c_A\f$ and \f$c_B\f$ are the input parameters center_A and
     133             :  * center_B, \f$r_A\f$ and \f$r_B\f$ are the input parameters radius_A and
     134             :  * radius_B, and \f$R\f$ is the outer boundary radius, we demand the
     135             :  * following restrictions on parameters:
     136             :  * - \f$c_A^0>0\f$; this is a convention to simplify the code.
     137             :  * - \f$c_B^0<0\f$; this is a convention to simplify the code.
     138             :  * - \f$|c_A^0|\le|c_B^0|\f$. We should roughly have \f$r_A c_A^0 + r_B c_B^0\f$
     139             :  *   close to zero; that is, for BBHs (where \f$r_A\f$ is roughly twice the
     140             :  *   mass of the heavier object A, and \f$r_B\f$ is roughly twice the mass
     141             :  *   of the lighter object B) the center of mass should be roughly
     142             :  *   at the origin.
     143             :  * - \f$0 < r_B < r_A\f$
     144             :  * - \f$R \ge 3(|c_A^0|-|c_B^0|)\f$; otherwise the blocks will be too compressed
     145             :  *   near the outer boundary.
     146             :  *
     147             :  */
